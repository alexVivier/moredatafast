# Widget Library

A gallery of every chart type MDF supports, each wrapped in the standard widget chrome.

## Components
- `Gallery.jsx` — single-file showcase using shared chart/feed components from `../dashboard/`
- Adds: `BigNumber`, `BarChart`, `Scatter`, `Heatmap`, `Sankey`, `Candle`, `MiniMap`

## Covered chart types
Line/area · Bar · Scatter · Donut · Heatmap · Sankey · Candlestick · Choropleth map · Lists (cities, events) · KPIs.
